# How to Export Lighthouse Report

## üìä Exporting Lighthouse Report

### **Method 1: Export as JSON (Best for Analysis)**

1. Run Lighthouse analysis
2. After the report loads, look for **"Export"** or **"Download"** button
3. Click it and select **"Save as JSON"**
4. Save the file (e.g., `lighthouse-report.json`)
5. Share the JSON file with me

### **Method 2: Export as HTML (Visual Report)**

1. Run Lighthouse analysis
2. Click **"Export"** or **"Download"** button
3. Select **"Save as HTML"**
4. Save the file (e.g., `lighthouse-report.html`)
5. You can open it in a browser to view
6. Share the HTML file with me

### **Method 3: Copy Report URL**

1. After Lighthouse runs, look for a **"Share"** or **"Copy Report URL"** button
2. Copy the URL
3. Share it with me

## üéØ Step-by-Step

### **Using Direct URL Method:**

1. Open `chrome://lighthouse/`
2. Enter: `http://localhost:5173`
3. Select categories
4. Click **"Analyze page load"**
5. Wait for report to generate
6. Look for **"Export"** button (usually top-right)
7. Click **"Export"** ‚Üí **"Save as JSON"** or **"Save as HTML"**

### **Using DevTools Method:**

1. Right-click ‚Üí Inspect ‚Üí Lighthouse tab
2. Run analysis
3. After report loads, click **"Export"** button (top-right of report)
4. Save as JSON or HTML
