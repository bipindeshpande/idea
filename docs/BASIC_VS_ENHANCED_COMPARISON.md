# Basic vs Enhanced Analysis Comparison

## Simple 10-Line Table

| **Section** | **Basic (90s)** | **Enhanced (+30-40s)** |
|-------------|-----------------|------------------------|
| **Financial Outlook** | Generic cost ranges ($6K-$19K) | Specific costs, unit economics (CAC, LTV, ratios) |
| **Risk Radar** | Generic risks ("market saturation") | Specific risks tied to idea + user constraints |
| **Roadmap** | Generic steps ("validate assumptions") | Specific tools, platforms, actions per phase |
| **Competitive Analysis** | Basic mentions | Deep competitor comparison + positioning |
| **Market Intelligence** | Basic market size | Entry strategy, timing, growth projections |
| **Success Metrics** | None | KPIs for each phase (0-30, 30-60, 60-90 days) |
| **Tool Stack** | None | Specific tools/platforms for each idea |
| **Validation Plan** | Basic questions | Prioritized validation roadmap with sequencing |
| **Risk Prioritization** | None | Risk impact matrix + monitoring strategies |
| **Value** | Good | **2-3x more actionable, specific, valuable** |

## Key Differences

**Basic**: Generic, ranges, "one-size-fits-all"  
**Enhanced**: Specific, personalized, actionable, idea-specific
